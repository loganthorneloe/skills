# Skill quality glossary

Load only when a term in SKILL.md needs depth.

## Predictability
Same *process* each run, not same tokens. Root virtue.

## Context load
Cost of always-loaded description (model-invoked).

## Cognitive load
Cost on human to remember user-invoked skills exist.

## Context pointer
Wording that tells agent *when* to open linked material. Weak wording = variance bug; fix wording before inlining.

## Completion criterion
Checkable "done" for a step. Vague → premature completion.

## Leading word
Compact pretrained concept anchoring behavior in few tokens.

## Seam
Public boundary to test/observe; not internals.

## Tracer bullet
Thin vertical slice, demoable end-to-end, one context window.
