# Failure → lever

| Signal | Lever |
|--------|-------|
| Never fires | Sharpen description / leading word |
| Fires too often | Narrow description or user-invoked |
| Stops early | Completion criteria; split steps |
| Repeat user correction | One SSOT rule; delete dupes |
| Ignores section | Move up ladder; fix pointer wording |
| Tool thrash | Tighter steps or script |
| Multi-step, no skill | New skill → `skill-mine` |
| Long unused | Deprecate/merge |
